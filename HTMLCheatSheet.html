Structural Markup:

  • Paragraph - < p > (Creates a new paragraph of text)
  • Bold - < b >
  • Italics - < i >
  • Superscript - < sup >
  • Subscipt - < sub >
  • Newline - < br / > (starts a new line)
  • Horizontal Line - < br / > (creates a horizontal line)

Semantic Markup:

  • < strong > - for important content
  • < em > - for ephmasized content
  • < blockquote > - for longer quotes (indent)

    HTML:
    < blockquote > This is the blockquote. < /blockquote >

    Actual:

    This is the blockquote.

  • < q > - for shorter quotes [NOT USED]
  • < abbr > - to expand upon abbreviations (when hovering over them)
  • < cite > - for refrences to pieces of work (such as books)
  • < dfn > - used when defining a new term
  • < address > - used for contacting the webpage author

    HTML:

    < p > 000 Palace Place, HTML < /p >
    < /address >

    000 Palace Place, HTML

  • < ins > - for whats been inserted
  • < del > - for whats been deleted
  • < s > - no longer relevant but not deleted

Lists:

Ordered Lists:

Lists where elements are sequentially ordered.

  • Opened with < ol > closed with < /ol >
  • Each element opened with < li > closed with < /li >

Example:

Race Results

  1. Billy
  2. Alice
  3. David

Unordered Lists:

Lists where elements are unordered.

  • Opened with < ul > closed with < /ul >
  • Each element opened with < li > closed with < /li >

Example:

Shopping List

  • Eggs
  • Apples
  • Avacados

Definition Lists:

Lists where elements are being defined.

  • Opened with < dl > closed with < /dl >
  • Each element to be defined is opened with < dt > closed with < /dt >
  • Each definition of an element is opened with < dd > closed with < /dd >

Example:

Nested Lists:

You can nest lists by opening a new list inside the element of another list.

Links:

Links are created using < a href="weblink" > TEXT < /a >

Other Sites:

HTML:

< a href="http://www.imdb.com" > IMDB < /a >

Actual:

IMDB

Same Site Pages:

Use the realitive path of the page

HTML:

< a href="OtherTestPage.html" > Other Test Page < /a >

Actual:

Other Test Page

Link Notes:

  • "mailto:" - add this to the front of the href string if its an email
    • HTML: < a href="mailto:boi@boi.boi" > boi@boi.boi < /a >
    • Actual: boi@boi.boi

  • "target" - add this as an attribute with the value "_blank"
    • HTML: < a href="http://www.imdb.com" target="_blank" > IMDB < /a >
    • Actual: IMDB

  • Linking to a specific part of a page: Must give a header an attribute: id="idname"
    • HTML: < a href="#top" > Top < /a >
    • Actual: Top

  • To link to a specific part of a different page, add #linkName to the end of the link
    • Ex: Hit the second link on this page...

Tables:

Table Tags:

  • < table > - contains the table data
  • < tr > - contains the data of a single row of the table
  • < td > - contains the data of a single cell of a row of the table
  • < th > - contains the heading for a single column of the table

Table Tag Attributes:

  • colspan (td/th) - how many column cells this cell will encompass
  • rowspan (td) - how many row cells this cell will encompass

Long Table Tags:

  • < thead > - headings of the table
  • < tbody > - body of the table
  • < tfoot > - footer of the table

Example:
Heading 1 Heading 2 Heading 3
Row 1 data1.(1/2) data1.3
Row 2 data(2/3).1 data2.2 data2.3
Row 3 data3.2 data3.3
Foot Row (4) data4.1 data4.2 data4.3
